# Code of Conduct

## Our pledge

To foster an open and welcoming environment, we are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- use respectful and inclusive language
- accept constructive feedback
- prioritize project and community health
- show empathy across different experience levels

## Unacceptable behavior

- personal attacks, harassment, or discrimination
- publishing private information without consent
- offensive, intimidating, or degrading language
- any inappropriate conduct in project spaces

## Enforcement

Maintainers may remove, edit, or reject contributions and interactions that violate this code.

## Reporting incidents

To report violations, open a private issue when available or use the channel listed in `SECURITY.md`.

## Scope

This code applies to all project spaces, including issues, pull requests, discussions, and associated channels.
